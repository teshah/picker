# Project Setup

To build and run the project, follow these steps:
1. Install React
   ```sh
   npm install -g pnpm
   pnpm add @vitejs/plugin-react --save-dev
   ```

2. Build the project using Vite:
   ```sh
   npx vite build
   ```

3. Run it
   ```sh
   npm run dev
   or
   pnpm run dev
   ```
